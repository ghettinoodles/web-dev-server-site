Key Terms

CERN: Cern is the company Tim Berners-Lee, the creator of the first web server, worked for.

Client: The hardware or software that initiates requests and consumes services from a server.

Data Centers: Facilities that house multiple web servers, typically between hundreds and thousands.

DNS: The phonebook of the internet, which translates human-friendly domain names into computer-readable IP addresses.

IP Address: A unique numerical label assigned to every device connected to a computer network that uses Internet Protocol for communication.

Latency: The time it takes data to travel back and forth.

Request response cycle: The fundamental, stateless communication pattern of the web where a client sends an HTTP request to a server, which processes it and returns a response.

Tim Berners-Lee: Inventor of the web server.

Web Server: A computer (hardware) and specialized software that stores web content—such as HTML documents, images, and CSS files—and delivers it to users over the internet.
